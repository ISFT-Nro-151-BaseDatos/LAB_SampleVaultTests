Para que el sistema funcione correctamente, seguí este orden lógico (Cadena de Dependencias):

1. INFRAESTRUCTURA (Manual):
   - Abrir el panel de UniServerZ.
   - Iniciar el servicio MySQL/MariaDB (Puerto 3306).
   - Iniciar el servicio Apache (Necesario para phpMyAdmin y gestión local).

2. APLICACIÓN (Automatizado):
   - Una vez que los servicios de UniServerZ estén en VERDE, ejecutar el archivo 'SampleVaultTests.bat'.

3. CIERRE DEL SISTEMA:
   - Para apagar el servidor, cerrá la ventana de la terminal o presioná Ctrl+C.
   - No olvides detener los servicios en UniServerZ para liberar recursos de memoria.

NOTA: Si el .bat falla, verificar que el archivo .env siga existiendo en la carpeta /backend.
